# PRODIGY_ML_03

## Task-03: Cats vs Dogs Image Classification

### Objective
Build an image classification model to distinguish between cats and dogs using a Convolutional Neural Network (CNN).

### Technologies Used
- Python
- TensorFlow
- Keras
- NumPy

### Dataset
Cats and Dogs Image Dataset

### Steps Performed
1. Loaded cat and dog image dataset
2. Rescaled image pixel values
3. Created training and validation datasets
4. Built a CNN model using TensorFlow/Keras
5. Trained the model for 5 epochs
6. Evaluated model performance
7. Saved the trained model

### Result
- Training Accuracy: 80.00%
- Validation Accuracy: 63.00%

### Project Files
- cat_dog_classification.py
- dataset/
- README.md

### Outcome
Successfully trained a CNN model for classifying cat and dog images.
